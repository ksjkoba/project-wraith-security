# Project Wraith: Automated API Security Auditor

Wraith is a specialized security tool designed to audit API endpoints for vulnerabilities, specifically targeting Identity and Access Management (IAM) flaws.

## Core Capabilities
* **Auth Bypass Testing:** Automated checks for OIDC/SAML misconfigurations.
* **Footprinting:** Discovers exposed endpoints and digital footprints in Windows environments.
* **Logic Auditing:** Tests for broken object-level authorization (BOLA) in IAM-protected APIs.

## Disclaimer
This tool is for educational and authorized security testing only.
